# 舒尔特方格 · 本地练习小程序

专注力训练工具，支持电脑浏览器和微信扫码访问。

## 快速开始

1. 双击 `start.bat` 启动服务
2. 浏览器会自动打开 `http://localhost:3456`
3. 手机连接同一 WiFi，用微信扫描页面上的二维码即可进入

## 功能

- 3×3 到 7×7 多种方格大小
- 数字模式 / 字母模式
- 实时计时与进度显示
- 最佳成绩、练习次数、平均用时（本地保存）
- 移动端适配，微信内置浏览器可直接使用

## 注意事项

- 手机和电脑必须在同一 WiFi 网络
- 若扫码无法访问，请检查 Windows 防火墙是否允许 Node.js 入站连接
- 关闭命令行窗口即停止服务

## 手动启动

```bash
npm install
npm start
```
